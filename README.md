
## 📌 프로젝트 Body & Hand 웹사이트

![01](https://github.com/user-attachments/assets/b2347b53-d12d-416a-8575-02d3226092ec)



▷ 개인 포트폴리오 입니다.

▷ 정보의 과잉으로 인해 브랜드의 핵심 가치가 희석된 기존 바디케어 웹사이트를 개선하기 위해 기획하게 되었습니다.
